# Python Practice Programs

This directory contains Python solutions and practice programs developed as part of the **60-Day Core Python Logic Challenge**.

## Overview

The goal of this collection is to improve logical thinking, master Python internals, and prepare for technical interviews. The scripts cover a wide range of topics defined in the challenge roadmap.

### Key Topics
*   **Logic Foundations:** Basic algorithms, loops, and conditionals.
*   **Data Structures:** Lists, dictionaries, strings, and custom implementations.
*   **Advanced Python:** Context managers (class & generator based), decorators, and recursion.
*   **Interview Questions:** Common problems like prime checks, non-repeating characters, and list reversals.

## Structure

*   **Roadmap:** The detailed schedule for the challenge is located in `🚀 60-Day Core Python Logic Challenge.txt`.
*   **Solutions:** Individual Python files are generally named with a number prefix or a descriptive title corresponding to the specific challenge (e.g., `50_Context manager (generator).py`).

## How to Run

You can run any specific practice program using the Python interpreter:

```bash
python "filename.py"
```

Example:
```bash
python "8_Prime_number_check.py"
```
